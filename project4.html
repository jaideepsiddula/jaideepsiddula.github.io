We can count the number of times each word shows up once a text is tokenised.This is called bag of words model and then we can look up at the sentiment value for each word from a sentiment lexicon that has it all pre recorded to classify the total sentiment value of our tweet .the code requires twitter's APIs, dependencies and then we write our sentioment analyser python script.Now, an API is a gateway that lets you access some servers internal functionality.
